Modern ve estetik bir arayüze sahip, Flutter ile geliştirilmiş burç uygulaması.  
Kullanıcılar tüm burçları grid şeklinde görüntüleyebilir ve her burcun detaylı özelliklerine modern bir detay sayfası üzerinden ulaşabilir.

## Özellikler

- 12 burcun tamamı
- Her burç için detaylı açıklamalar (10+ özellik)
- Modern grid ana ekran
- Poster tarzı detay sayfası
- Glassmorphism (blur) tasarım
- Draggable detay panel
- Burca özel renk teması
- Smooth scroll ve animasyonlar
- Tamamen responsive UI

##  Kullanılan Teknolojiler

- Flutter
- Dart
- Material Design
- Custom UI / UX
- Sliver & DraggableScrollableSheet

##  Kurulum

Projeyi çalıştırmak için:


git clone https://github.com/kullanici-adi/horoscope_app.git
cd horoscope_app
flutter pub get
flutter run
